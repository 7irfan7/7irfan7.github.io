Team Project: Cryptocurrency Website Design

  • Category: Web design
  • Project date: 2023

This project is a group assignment for the Human and Computer Interaction (IMK) course, which focuses on user interface (UI) and user experience (UX) design for a digital money website. In this project, the three of us were given the freedom to choose the topic and design concept, with the main goal of creating a responsive and easy-to-use web interface. The website we designed aims to provide an intuitive and user-friendly platform for cryptocurrency users. The main features we have developed include: User Dashboard: Presents users' financial information concisely, including balances, recent transactions, and changes in asset values. Digital Money Transactions: Make it easy for users to send and receive cryptocurrency with a simple and secure interface. Charts and Statistics: Displays real-time market data and cryptocurrency price changes with informative visualizations. Security and Privacy: Emphasizes security features, such as two-factor authentication and data encryption This project will be presented to the class, where we will explain the design process, the challenges faced, and the solutions we implemented to create an effective interface. We will also show a demo of the website design that we have created.
